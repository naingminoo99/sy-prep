Phase 1 · Weeks 1–12

Foundations 🌱

From absolute zero to your first ML project. Tick each week once you've finished all the tasks — progress saves automatically!

Week 1

Python setup + variables

  • Install Anaconda and open Jupyter Notebook
  • Enroll in Python for Everybody on Coursera
  • Complete variables, expressions & data types
easy coursera
Week 2

Conditionals + loops

  • Learn if / else, for loops, while loops
  • Write a script to print even numbers
  • Learn how to define and call functions
easy
Week 3

Strings + data structures

  • Learn strings and lists (indexing, slicing)
  • Learn dictionaries and tuples
  • Exercise: countries and capitals dictionary
easy intermediate
Week 4

Python wrap-up

  • Finish all Coursera course assignments
  • Do 5 HackerRank exercises
  • Enroll in Applied Data Science specialization
intermediate coursera
Week 5

Pandas basics

  • Learn DataFrames and Series
  • Practice loading and exploring a CSV dataset
  • Filter rows and select columns
intermediate
Week 6

NumPy + data cleaning

  • Learn NumPy arrays and operations
  • Handle missing data — dropna, fillna strategies
  • Compare DataFrame vs NumPy — when to use which
intermediate
Week 7

Data visualisation

  • Learn matplotlib — line, bar, scatter, histogram
  • Plot a histogram and scatter chart from real data
  • Learn seaborn heatmaps for correlation
intermediate
Week 8

Intro to Machine Learning

  • Enroll in ML Specialization (Andrew Ng)
  • Learn supervised vs unsupervised learning
  • Understand linear regression basics
intermediate coursera
Week 9

Linear regression

  • Understand gradient descent intuitively
  • Build a regression model with scikit-learn
  • Evaluate with MSE, RMSE, and R²
hard
Week 10

Classification

  • Learn logistic regression — concept + sklearn
  • Learn overfitting and regularisation (L1/L2)
  • Build a Titanic survival classifier
hard
Week 11

Neural networks intro

  • Learn neural network basics — neurons, layers, activation
  • Complete ML Specialization assignments
  • Save and celebrate your certificate! 🎉
hard
Week 12

Project 1 — Kaggle Titanic 🚢

  • Complete the Kaggle Titanic challenge end-to-end
  • Write a clear project summary — what did you learn?
  • Upload notebook to GitHub (your first repo!)
hard project

Phase 2 · Weeks 13–24

Intermediate 🔮

Going deeper — pipelines, SQL, ensemble models, NLP, and time series. The skills that will set you apart in the programme.

Week 13

Sklearn pipelines

  • Learn sklearn Pipeline — what it is and why it matters
  • Chain preprocessing + model into one pipeline
  • Learn feature scaling — StandardScaler, MinMaxScaler
intermediate
Week 14

Cross-validation & tuning

  • Learn k-fold cross-validation and why it's essential
  • Apply cross_val_score to a real model
  • Learn GridSearchCV for hyperparameter tuning
intermediate hard
Week 15

Trees + ensembles

  • Build a decision tree and visualise it
  • Learn random forest — how bagging helps
  • Evaluate with confusion matrix, precision, recall
hard
Week 16

SQL basics

  • Learn SELECT, WHERE, ORDER BY, LIMIT
  • Learn GROUP BY, HAVING, and aggregate functions
  • Practice on SQLZoo or Mode Analytics
intermediate
Week 17

Advanced SQL

  • Learn INNER JOIN, LEFT JOIN, RIGHT JOIN
  • Write subqueries and CTEs (WITH clause)
  • Learn window functions — RANK, ROW_NUMBER, LAG
intermediate hard
Week 18

Feature engineering

  • Create new features from existing columns
  • Encode categoricals — LabelEncoder, get_dummies
  • Polynomial features and interaction terms
hard
Week 19

Gradient boosting (XGBoost)

  • Understand boosting — how it differs from bagging
  • Train an XGBoost model and compare to random forest
  • Tune with GridSearchCV; check feature importances
hard
Week 20

Unsupervised learning

  • Learn K-means clustering — how it works intuitively
  • Apply PCA for dimensionality reduction
  • Visualise clusters with scatter plots
hard
Week 21

NLP basics

  • Learn text cleaning — lowercase, stopwords, punctuation
  • Learn tokenisation and TF-IDF vectorisation
  • Build a simple sentiment classifier
hard
Week 22

Time series

  • Work with datetime in pandas — parse, resample, shift
  • Compute rolling averages and seasonal decomposition
  • Forecast with a simple ARIMA or linear trend model
hard
Week 23

Deep learning with Keras

  • Build a multi-layer neural network in Keras
  • Understand epochs, batch size, and learning rate
  • Train on MNIST and hit >97% accuracy
hard
Week 24

Project 2 — Customer churn 📊

  • Full pipeline: EDA → clean → feature engineer → model
  • Compare logistic regression, random forest, XGBoost
  • Write a summary notebook and upload to GitHub
hard project

Phase 3 · Weeks 25–36

Application 🚀

Advanced models, deployment, ethics, and the capstone project. This is where everything comes together for your MTech application.

Week 25

Convolutional networks (CNN)

  • Understand CNN architecture — filters, pooling, feature maps
  • Build a CNN in Keras for image classification
  • Train on CIFAR-10 and analyse results
hard
Week 26

Transformers & modern NLP

  • Understand attention and transformers intuitively
  • Use HuggingFace to run a pre-trained BERT model
  • Fine-tune on a text classification task
hard
Week 27

Model deployment basics

  • Save and load a trained model (pickle / joblib)
  • Build a simple prediction API with FastAPI
  • Deploy to a free cloud service (Render or Hugging Face Spaces)
hard
Week 28

Cloud & MLOps basics

  • Intro to Google Cloud / AWS — what they offer for ML
  • Learn experiment tracking with MLflow or Weights & Biases
  • Understand model versioning and reproducibility
hard
Week 29

Ethics & responsible AI

  • Understand bias and fairness in ML models
  • Learn explainability with SHAP values
  • Read one ethics in AI paper and summarise it
intermediate
Week 30

Research paper reading

  • Read "Attention Is All You Need" (Transformers paper)
  • Read one recent AI application paper in your area of interest
  • Write a 1-page plain-English summary of each
hard
Week 31

Capstone planning 🗺️

  • Choose a meaningful real-world problem to solve
  • Find and explore a suitable dataset
  • Define success metrics and write a 1-page project plan
hard project
Week 32

Capstone — EDA & cleaning

  • Full exploratory data analysis with visualisations
  • Clean and preprocess the dataset thoroughly
  • Document every decision with written observations
hard project
Week 33

Capstone — modelling

  • Build at least 3 different models and compare them
  • Tune the best model with cross-validation
  • Interpret results and document findings
hard project
Week 34

Capstone — polish & deploy

  • Refine the notebook — clean code, clear markdown
  • Deploy model as a simple web demo
  • Record a short video walkthrough of the project
hard project
Week 35

Portfolio & GitHub polish

  • Clean up all 3 project repos with proper READMEs
  • Create a pinned GitHub portfolio with project previews
  • Update LinkedIn with skills, projects, and certificates
intermediate
Week 36

MTech application prep ✨

  • Write and refine your Statement of Purpose
  • Prepare answers to common interview questions
  • Final review — you made it! Send the application 🌸
intermediate milestone

Courses & tools

Free Python notebook in your browser — no installation, runs on Google's servers. The easiest way to get started.

The best beginner Python course on Coursera. Friendly, clear, and completely beginner-safe. Start here on Week 1.

Covers pandas, visualisation, and applied ML. Enroll in Week 4 once you've got the Python basics down.

ML · Week 8+

ML Specialization (Andrew Ng)

The gold standard ML course. Andrew Ng explains everything with intuition first — no scary maths required.

Free bite-sized ML courses with interactive notebooks. Use Titanic and House Prices as your first projects.

Free interactive SQL practice with real queries. SQLZoo is great for beginners; Mode has more advanced challenges.

The go-to library for transformers and modern NLP. Free models, datasets, and Spaces for deploying demos.
